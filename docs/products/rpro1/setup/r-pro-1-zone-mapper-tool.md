# R-PRO-1 Zone Mapper tool

The Apollo R-PRO-1's LD2450 mmWave sensor tracks up to three targets across the room. With the Zone Mapper tool you draw its detection zones on a visual grid in Home Assistant, watch targets move in real time, and get an occupancy sensor for every zone you draw.
